# Everytime / 에브리타임

에브리타임은 대학생을 위한 커뮤니티입니다.   
메인 화면에서 회원 정보 조회 및 게시판들을 확인할 수 있습니다.   
각 게시판에서는 게시판의 성질에 맞게 익명, 비익명으로 게시글을 작성할 수 있습니다.
